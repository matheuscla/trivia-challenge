# Trivia Challenge

A Triva game developed using React Native.

## How to run
Clone the repo, run `yarn` to install the dependecies then you can run `yarn ios` or `yarn android` to open the simulator.

## Technologies and Libs
- Expo
- React Native
- Styled components
- Redux
- Redux Thunk
